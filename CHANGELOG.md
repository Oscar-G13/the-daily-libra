## [0.1.0] - 2026-03-21

### Added
- Initial project scaffold
- Next.js 14 App Router structure
- Supabase auth integration (email + Google OAuth)
- Onboarding flow with birth data intake and Libra archetype quiz
- Natal chart calculation engine
- AI reading engine (OpenAI GPT-4o) with 13 reading categories
- 5 reading style modes
- Personalized dashboard
- AI reflection companion with memory context
- Journal with mood tagging
- Stripe subscription integration (free + premium tiers)
- Dark editorial luxury design system
- Framer Motion animations
- PostHog analytics
